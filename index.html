Service

できる事

HTML / CSS

ホームページを制作するために必須となるマークアップ言語。コーディングの基本部分ですが、何事も「基本が一番大事」といわれるように、web制作においても最も重要な部分となります。

jQuery / JavaScript

動きのあるホームページを作るために必要なのがこの２つです。こちらのサイトでも下にスクロールすると、要素が動いているはこれらの言語を使っています。

Bootstrap

webサイトを効率よく作るためのwebフレームワークです。レスポンシブ対応されているので、PC、タブレット、スマホの画面サイズに自動的に対応したホームページをつくることができます。

Wordpress

WordPress(ワードプレス)でのホームページを約３年ほど運営しておりました。サイトの高速化やSEO対策となるプラグインを組み合わせて構築することができます。

SEO対策

ホームページはただ作れば良いというわけではありません。どれだけ見た目が美しいホームページでも、狙った検索ワードで上位表示されないとまったく意味のないものになります。
